¿Para qué es la donación?

Lista de apoyos (ejemplo)

Con el dinero recolectado se puede:

  • Comprar mercados para familias.

  • Apoyar actividades comunitarias.
  • Ayudar en emergencias básicas.

Definiciones rápidas

Donación:
Es un aporte voluntario de dinero o recursos para ayudar a una causa.
Causa:
Es el objetivo o motivo por el cual se recolectan las donaciones.
Transparencia:
Significa mostrar cómo se usa el dinero de manera clara y responsable.
Transparencia

Nos comprometemos a usar el dinero de forma responsable.
